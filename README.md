# Le Système Coach - Page de Vente

Page de vente premium pour "Le Système Coach" - Formation pour entraîneurs de football amateur.

## 🚀 Déploiement sur Vercel

### Méthode 1 : Via GitHub (Recommandé)

1. Crée un repo GitHub
2. Push ce code
3. Connecte le repo à Vercel
4. Vercel déploie automatiquement

### Méthode 2 : CLI Vercel

```bash
npm install -g vercel
vercel
```

### Méthode 3 : Drag & Drop

1. Crée un ZIP du dossier
2. Va sur vercel.com
3. Drag & drop le ZIP

## 📦 Installation locale

```bash
npm install
npm run dev
```
